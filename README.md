# 🌤️ Weather Dashboard (React)

A modern, responsive, and visually enhanced Weather Dashboard built using React.  
This app allows users to search any city and view real-time weather details with a clean and premium UI.

🔗 **Live Repository:**  
https://github.com/shubhamkoti/weather-dashboard-react

---

## 🚀 Features

- 🔍 Search weather by city name
- 🌍 Get real-time weather data
- 🌡️ Detailed information:
  - RealFeel temperature
  - Humidity
  - Cloud coverage
  - Min / Max temperature
- ⏰ Hourly forecast section
- 🌙 Light / Dark mode toggle
- 🎨 Modern UI with glassmorphism & gradients
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 CSS / Tailwind CSS
- 🌐 Weather API (OpenWeather or similar)
- 💡 JavaScript (ES6+)
